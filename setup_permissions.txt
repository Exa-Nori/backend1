ИНСТРУКЦИИ ПО УСТАНОВКЕ ПРАВ ДОСТУПА

1. ЧЕРЕЗ FTP КЛИЕНТ:
   - Файлы .py: 755 (выполняемые)
   - Файлы .txt, .pyc: 644 (только чтение)
   - Папки: 755 (выполняемые)
   - .htaccess: 644

2. ЧЕРЕЗ ФАЙЛОВЫЙ МЕНЕДЖЕР ХОСТИНГА:
   - Правый клик на файл → Права доступа
   - Python файлы: 755
   - Остальные файлы: 644
   - Папки: 755

3. ЧЕРЕЗ SSH (если доступен):
   chmod 755 *.py
   chmod 644 *.txt
   chmod 644 .htaccess
   chmod 755 public/
   chmod 644 public/*

4. ПОРЯДОК ТЕСТИРОВАНИЯ:
   1. Загрузите все файлы
   2. Установите права доступа
   3. Переименуйте .htaccess_simple в .htaccess
   4. Откройте test.py в браузере
   5. Если работает, переименуйте обратно основной .htaccess

5. ЕСЛИ НЕ РАБОТАЕТ:
   - Проверьте логи ошибок в панели хостинга
   - Убедитесь, что Python включен в настройках
   - Попробуйте разные версии .htaccess файлов
